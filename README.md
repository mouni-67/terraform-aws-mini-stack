# Terraform AWS Mini Stack

## What this project does
- Creates AWS VPC
- Creates Subnet, IGW, Security Group
- Deploys EC2 instance with nginx
- Creates S3 bucket

## How to run
terraform init
terraform plan
terraform apply

## Outputs
- EC2 public IP
- VPC ID
- S3 bucket name

## Design
- Modular Terraform structure
- Separated network and compute
- Easy scalable design
